# Авария "Температура канала выше температуры продукта"

# 1. Краткое описание:

Тип склада: для обычных и комбинированных
Возникает, если склад работает в режиме хранения, приточный клапан открыт более 5 мин и мининимальная темппература канала выше чем минимальная температура продукта, через 1 час авария автоматически сбрасывается. Аварийное сообщение показывается на страницах "Сигналы", "Секции", "Секция Х". Склад закрывает все клапаны и выключает вентиляторы.

# 2. Условия:

-   Склад включен
-   Склад в режиме хранения
-   Секция в авто режиме
-   Приточные клапана исправны

# 3. Путь к настройкам

Настройки -> Охлаждение -> Температура канала выше температуры продукта

# 4. Описание настроек

1. Время слежения - если условия для аварии сохраняются в течении данного времени, то установится авария (заводское значение: 5 мин)
2. Время автосброса аварии - время, по истечению которого авария автоматически сбросится (заводское значение: 1 час)

# 5. Описание (программист)

Путь `src\control\extra\extralrm\def\open_vin\index.js`
Код `openVin`
Принадлежность: для склада

1. Сначала проверяется разрешение на работу функции (проверка условий аварии: склад включен, в режиме хранения, есть хотя бы 1 секция в авто, имеются настройки, имеется приточный клапан, приточный клапан открыт и темп канала выше темп продукта)
2. При соблюдении условий засекается таймер на "Время слежения", по истечению данного таймера и условия аварии не изменились, устанавливается авария
3. Засекается таймер сброса аварии.
4. Сброс аварии: по таймеру, по нажатию кнопки, нет секции в авто режиме, выключили склад, изменили режим склада отличный от Хранения, обнулили настройки.
