# Авария "Превышено время работы с закрытыми клапанами"

# 1. Краткое описание:

Тип склада: для обычных и комбинированных в режиме обычного
Возникает, при работе напорных вентиляторов секции на закрытые клапана больше чем заданное время в настройках. Аварийное сообщение показывается на страницах "Сигналы". Склад закрывает все клапаны и выключает вентиляторы.

# 2. Условия:

-   Склад включен
-   Секция в авто режиме
-   Нет настроек
-   Приточные клапана исправны
-   Настройки сушки "Постоянный вентилятор": Выкл
-   Настройка вентиляции "Управление вентиляцией": Авто, вкл

# 3. Путь к настройкам

Настройки -> Работа клапанов

# 4. Описание настроек

1. Разрешенное время закрытых приточных клапанов - по истечению данного времени, при работающих напорных ВНО, возникнет авария(заводское значение: 15 мин)
2. Время ожидания сброса аварии - время, по истечению которого авария автоматически сбросится (заводское значение: 30 мин)

# 5. Описание (программист)

Путь `src\control\extra\extralrm\def\overVlv\index.js`
Код `overVlv`
Принадлежность: для каждой секции

1. Сначала проверяется разрешение на работу функции.
2. Вычисляется условие аварии (ВНО секции запущены, приточный клапан секции закрыт), засекается время "Разрешенное время закрытых приточных клапанов"
3. По истечению времени, если условие аврии не ушло, то устанавливает авария
4. Авария появилась, засекается таймер сброса аварии.
5. Сброс аварии: по таймеру, по нажатию кнопки, переключение режима секции с авто на что-либо, выключение склада, обнуление настроек, Склад в режиме сушки и включили ВНО на постоянную работу, измененеие режима внутренней вентиляции на Вкл, комбинированный склад перешел в режим холодильник.
